# Guitar EAD Chat - Frontend Client

This repository hosts the Flutter Web Client for the Real-Time Audio Platform PoC.

## Gitflow & Branching Strategy
We use Gitflow with the naming conventions:
- `feature/<feature-name>`
- `fix/<bug-description>`
- `hotfix/<fix-description>`

## Setup & Running
1. Run `flutter pub get`
2. Run `flutter run -d chrome`
